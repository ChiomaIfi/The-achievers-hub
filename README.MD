# The Achievers Hub

This repository contains the source code for **The Achievers Hub**, a static website built and hosted using **GitHub Pages**.

The site is organized using a folder-per-page structure, where each directory represents a page on the website and contains an `index.html` file.

## Pages
- Home
- About
- Academics
- Contact
- The Achievers Hub

Additional sections may be added as the site grows.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Deployment
This site is automatically deployed using **GitHub Pages**.  
Any changes pushed to the `main` branch are reflected on the live site.

## Notes
This repository is intended for website development and maintenance only.
